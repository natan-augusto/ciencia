# Quadratic Equation Solver v2.0

A web-based quadratic equation solver derived from an original Python implementation, developed with AI assistance.

## 📖 About

This application solves quadratic equations of the form `ax² + bx + c = 0` (where `a ≠ 0`). It calculates the discriminant and finds the real roots when they exist.

The application runs entirely in the browser and mimics the exact behavior and output format of the original Python terminal program.

## 🐍 Original Python Version

This web application is a direct adaptation of my original Python code. The logic, output formatting, and user interaction flow have been preserved to maintain the same experience.

**Original Python features preserved:**
- Sequential input (a → b → c)
- Real-time validation (a ≠ 0)
- Colored output (green for success, red for errors)
- 1-second delays between calculations (simulating `time.sleep(1)`)
- Automatic loop for multiple equations
- Period (`.`) to comma (`,`) conversion for decimal numbers

## AI-Assisted Development

This HTML/JavaScript version was developed with the aid of **Artificial Intelligence** as a learning exercise to:

AI assisted in:
